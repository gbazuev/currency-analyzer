                        Добро пожаловать в CurrencyAnalyzer
                Это универсальный инструмент для первичного анализа котировок валют

                        На выбор доступно несколько команд:
                --get - инициирует API-запрос в Центробанк
                --exit - останавливает работу парсера
                --help - выводит на экран данное FAQ

                        Доступно несколько расширений для --get
                --today: - выводит котировки всех доступных валют на ближайшую дату
                --today: CURRENCY_NAME, ... - выводит котировки валюты/некоторых валют на ближайшую дату

                --date: dd/mm/yyyy - выводит котировки абсолютно всех доступных валют на заданную дату
                --date: dd/mm/yyyy CURRENCY_NAME, ... - выводит котировки валюты/валют на заданную дату

                --interval: dd1/mm1/yyyy1, dd2/mm2/yyyy2, CURRENCY_NAME - выводит котировку валюты на заданный промежуток
